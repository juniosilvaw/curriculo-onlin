# 💻 Currículo Online — Anderson Júnior

Site de currículo/portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro.

🔗 **[Ver online →](https://juniosilvaw.github.io/curriculo-online)**

---

## 🛠 Tecnologias

- HTML5 semântico
- CSS3 (variáveis, grid, flexbox, animações)
- JavaScript (Intersection Observer API)
- Google Fonts (Inter + IBM Plex Mono)

## 📁 Estrutura

```
curriculo-online/
└── index.html   # Arquivo único — todo o site
```

## 🚀 Como usar localmente

```bash
# Clone o repositório
git clone https://github.com/juniosilvaw/curriculo-online.git

# Abra o arquivo no navegador
cd curriculo-online
open index.html   # macOS
# ou apenas arraste o index.html para o navegador
```

## 🌐 GitHub Pages

Este site está publicado via GitHub Pages. Para publicar o seu:

1. Vá em **Settings** do repositório
2. Clique em **Pages** (menu lateral)
3. Em *Source*, selecione a branch `main` e pasta `/ (root)`
4. Salve — em alguns minutos o link estará disponível
